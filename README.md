# 📊 Portafolio de Análisis de Datos | Luis Carlos Pallares

¡Bienvenido a mi repositorio! Este es el código fuente de mi sitio web personal y portafolio profesional, donde comparto proyectos de análisis de datos, visualización, automatización y ciencia de datos.

🌍 **Puedes visitar el sitio web en vivo aquí:** [lcpallares.github.io](https://lcpallares.github.io/)

## 🚀 Sobre el Sitio Web

Este sitio está construido con **[Quarto](https://quarto.org/)**, un sistema de publicación científica y técnica de código abierto. Me permite documentar mis proyectos de manera ágil utilizando **Markdown** y ejecutar código directamente para mostrar análisis, gráficos interactivos y reportes detallados en un solo lugar.

El sitio está alojado de forma gratuita gracias a **GitHub Pages**.

## 📁 Estructura del Proyecto

La estructura principal de este repositorio de Quarto está organizada de la siguiente manera:

* `_quarto.yml`: Archivo de configuración global del sitio (menús, temas, estilos).
* `index.qmd`: Página de inicio del portafolio.
* `posts/` o `projects/`: Carpetas que contienen los análisis individuales, códigos y reportes en formato `.qmd` o `.ipynb`.
* `styles.css`: Personalizaciones de diseño para el sitio web.

## 🛠️ Cómo ejecutar o compilar este proyecto localmente

Si deseas clonar este repositorio y realizar pruebas locales, asegúrate de tener instalado **Quarto** y sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/lcpallares/lcpallares.github.io.git](https://github.com/lcpallares/lcpallares.github.io.git)
    cd lcpallares.github.io
    ```

2.  **Previsualiza el sitio localmente:**
    Este comando abre un servidor local con recarga en vivo para que veas los cambios en tiempo real:
    ```bash
    quarto preview
    ```

3.  **Compila el sitio web:**
    Para generar los archivos HTML estáticos finales:
    ```bash
    quarto render
    ```
